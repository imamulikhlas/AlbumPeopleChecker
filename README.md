## Tech Stack  
**Client:** Python v 3.11

## Run Locally  

Clone the project  

~~~bash  
  git clone https://github.com/imamulikhlas/AlbumPeopleChecker.git
~~~

Go to the project directory  

~~~bash  
  cd AlbumPeopleChecker/Python
~~~

Install Depedencies
~~~bash  
  pip install -r requirement.txt
~~~

Start the server  

~~~bash  
  py app.py
~~~
