## Run Locally  

Clone the project  

~~~bash  
  git clone https://github.com/imamulikhlas/AlbumPeopleChecker.git
~~~

Go to the project directory  

~~~bash  
  cd AlbumPeopleChecker
~~~

Start the server  

~~~bash  
py app.py
~~~
